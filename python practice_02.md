* Python 數字類型轉換
```
•	int(x) 將x轉換為一個整數。
•	float(x) 將x轉換到一個浮點數。
•	complex(x) 將x轉換到一個複數，實數部分為 x，虛數部分為 0。
•	complex(x, y) 將 x 和 y 轉換到一個複數，實數部分為 x，虛數部分為 y。x 和 y 是數字運算式。
```
