# 1.一般Python的函式：
```
abs(x)：回傳x的絕對值_abs(-2)=2
max(x1,x2,...)：回傳x1,x2,...的最大值_max(100,200,900)=900
min(x1,x2,...)：回傳x1,x2,...的最小值_min(100,200,900)=100
pow(a,b)：回傳a的b次方_pow(10,3)=1000
round(x)：回傳最接近x的整數。若與兩整數相近則傳回偶數值_round(4.5)=4
round(x,n)：回傳捨位到小數點後n位的浮點數_round(9.265,2)=9.27
```
# 2.數學函式：
```
fabs(x)：以浮點數回傳x的絕對值_fabs(-8)=8.0
ceil(x)：回傳大於x的最小整數_ceil(6.1)=7
floor(x)：回傳小於x的最小整數_floor(5.2)=5
exp(x)：回傳e的x次方_exp(1)=2.71828
log(x)：回傳log以e為底的(x)_log(2.71828)=1.0
log(x,base)：回傳以指定基底的對數_log(100,10)=2.0
sqrt(x)：回傳√x_sqrt(9.0)=3
sin(x)：回傳以弧度為單位的sine三角函式_sine(3.14159)=1
asin(x)：回傳以弧度為單位的反sine三角函式_asine(1.0)=1.57
cos(x)：回傳以弧度為單位的cosine三角函式_cosine(3.14159)=-1
acos(x)：回傳以弧度為單位的反cosine三角函式_cosine(1.0)=0
tan(x)：回傳以弧度為單位的tangent三角函式_tan(0.0)=0
degrees(x)：將x的角度從弧度(radian)轉換為度數(degree)_degrees(1.57)=90
radians(x)：將x的角度從度數(degree)轉換為弧度(radian)_radians(90)=1.57
```
